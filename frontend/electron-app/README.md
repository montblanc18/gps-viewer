# electron-app

## Requirement

- ```node > 14.0.0```

## Project setup
```
npm install

## for mac
# failed build fsevent if you type npm install without CXX=clang++ option
CXX=clang++ npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
